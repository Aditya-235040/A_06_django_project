# WhatsApp Appointment Application

## 📌 Overview
This project is a web-based appointment booking system integrated with WhatsApp.  
It allows businesses to manage customer bookings, send reminders, and assign staff efficiently.

## ✨ Features
- 📅 Appointment scheduling with admin dashboard  
- 👥 Staff assignment and management  
- 🔔 WhatsApp reminders for customers  
- 📊 Reporting and analytics  
- 🔒 Secure authentication and user profiles  

## 🛠️ Tech Stack
- **Backend:** Django  
- **Frontend:** Bootstrap (responsive UI)  
- **Database:** SQLite / PostgreSQL  
- **Integration:** WhatsApp API
